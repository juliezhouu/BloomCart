# BloomCart Icons

This folder should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create these icons using:
1. **Online tools**: Use services like Canva, Figma, or Adobe Express
2. **Design software**: Photoshop, GIMP, or Sketch
3. **Icon generators**: Browser extension icon generators

## Design Guidelines

- Use a plant/flower/leaf theme to match BloomCart's sustainability focus
- Use green colors (#7CB342, #558B2F) to match the brand
- Keep the design simple and recognizable at small sizes
- Ensure good contrast for visibility
